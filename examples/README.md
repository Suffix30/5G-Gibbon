# Example Reports

These are sample reports generated by 5G-Gibbon to demonstrate the reporting capabilities.

## Files

| File | Description |
|------|-------------|
| `demo_topology.html` | Interactive network topology visualization with attack paths |
| `demo_report.html` | Comprehensive HTML security assessment report |
| `demo_report.json` | Machine-readable JSON export of findings |

## Viewing

**HTML Reports:** Open directly in any web browser - no server required.

**JSON Report:** View in any text editor or JSON viewer.

## Features Demonstrated

- Network node discovery and mapping
- Attack path visualization with directional arrows
- Compromised node highlighting
- Detailed forensic attack logs
- Interactive node details (click nodes for full info)
- PDF export capability
- Severity-based finding categorization

## Generate Your Own

```bash
# Generate reports from your scan session
python run.py report html
python run.py report json
python run.py report topology

# Or use the session report (includes all data from your session)
python run.py report session
```
